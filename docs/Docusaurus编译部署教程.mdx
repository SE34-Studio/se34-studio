---
title : Docusaurus编辑部署教程
---

# 1.调试运行
Docusaurus可以在本地Node.js环境中进行调试运行。只需要在项目根目录下执行以下命令即可：
```bash
pnpm start
```
# 2.编译
因为此项目使用TypeScript作为开发语言，所以在某些情况下，调试能通过不等于编译能通过。
在确定TypeScript代码无误之后，可以通过以下命令编译:
```bash
pnpm build
```
# 3.部署
Docusaurus可以将编译后的静态文件部署到任何静态文件服务器上。只需要将编译后的`build`目录下的文件上传到服务器即可。
